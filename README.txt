#OVERVIEW
React Jigsaw is a template for building customized static sites with react. it includes several custom components that can be jigsawed together to quickly build a static interface.

#More
The aim of this project is to allow users to build Static sites with React.
It doesn't attempt to overcomplicate by running a server in the background, providing fake APIs or a bunch of unnecessary NPM modules for routing and writing to a server.
Just static front-end components, light and responsive.
